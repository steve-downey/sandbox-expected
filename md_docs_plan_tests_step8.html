Test Plan: Step 8 — expected<T, E&> Error-Reference Specialization

Standard section: Novel beyond standard — mirror of Step 7, error side. Test file: tests/beman/expected/expected_ref_e.test.cpp Negative-compile files: tests/beman/expected/expected_ref_e_*_fail.cpp

Testing Strategy

Catch2. Include header twice. This specialization mirrors Step 7 but for the error type:

  • Value T is stored by value (same as primary template)
  • Error E is stored as a non-owning pointer E* with rebind semantics
  • error() returns E&
  • Assignment from unexpected<G> rebinds the error pointer, does not assign through
